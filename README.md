# 💰 Expense Tracker

A simple and responsive Expense Tracker built using **HTML, CSS, and JavaScript**.

This project helps users manage their daily expenses by allowing them to add, search, filter, and delete expenses while automatically calculating the total amount.

---

## 🚀 Features

- ➕ Add New Expense
- 🗑 Delete Expense
- 🔍 Search Expense
- 📂 Filter by Category
- 💵 Automatic Total Calculation
- 💾 Local Storage Support
- 📱 Responsive Design

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
